A simple example when there IS signal

What if we sampled

  • 10 observations from a normal(mean=0,sd=1.0) distribution

  • 10 observations from a normal(mean=1.5,sd=1.0)

Would a t-test be able to detect a significant difference between the two?

A simple example when there IS signal

What if we did this 1000 times?

  testSignal = function(...){
    x1 = rnorm(n=10, m=0, sd=1)
    x2 = rnorm(n=10, m=1.5, sd=1)
    res = t.test(x1, x2)
    res$p.value
  }

  N=1000
  pvals.sig = sapply(1:N, testSignal)
  hist(pvals.sig, breaks = seq(0,1, length=100))

  #how many results have a pval < 0.05?
  sum(pvals.sig < 0.05)
## [1] 874

A simple example when there IS signal

Out of 1000 tests we successfully rejected the null 874 times (true positives). A pretty powerful test and we found many significant results.

What if we repeated it 1000 times?

  testNoSignal = function(...){
    x1 = rnorm(n=10, m=0, sd=1)
    x2 = rnorm(n=10, m=0, sd=1)
    res = t.test(x1, x2)
    res$p.value
  }
  pvals.nosig = sapply(1:N, testNoSignal)
  hist(pvals.nosig, breaks = seq(0,1, length=100))

  #how many results have a pval < 0.05?
  n1 = sum(pvals.nosig < 0.05)
  print(n1)
## [1] 39

A simple example when there is NO signal

This time there are 39 tests where the null was incorrectly rejected (false positives). 39 out of 1000 is 3.9%, which is pretty much what we expect when we choose a significance level(alpha) of 0.05.

a simple example with both

In the real world, we dont know which datasets have signal and which do not. When going through results we want a selection criteria that minimizes the false positives, but is also able to find real signal. If we set our alpha too low (bonferroni) well miss out on a lot of the interesting results. If we dont do anything, well be dealing with a lot of garbage. FDR is a good compromise.

a simple example with both

Lets look at the histogram of the pvals when we mix the two datasets together.

  pvals.both = c(pvals.sig, pvals.nosig)
  hist(pvals.both, breaks=seq(0, 1, length=100))

a simple example with both

if we do nothing, and just select the results with an unadjusted pval < 0.05, well wind up with 913 positive results.

  • true positives: 874

  • false positives: 39

a simple example with both

if we use bonferonni, and just select the results with an unadjusted pval < 0.05/1000 = 0.00005, well wind up with 70 positive results.

  • true positives: 70

  • false positives: 0

a simple example with both

if we use FDR and select those with an adjusted qval < 0.05, well wind up with 784 positive results.

  • true positives: 768

  • false positives: 16

a simple example with both

In omics studies, and most studies in general, the signal were trying to find makes up a very small portion of the dataset were searching through. what happens when the ratio of signal/nosignal changes?

#return the numbers of true/false positives when selecting from 
#unadjusted, bonferroni, FDR
checkSignificance = function(N1 = 1000, N2 = 10000, alpha=0.05){
  pvals.sig = sapply(1:N1, testSignal)
  pvals.nosig = sapply(1:N2, testNoSignal)
  pvals.both = c(pvals.sig, pvals.nosig)  
  qvals.both = p.adjust(pvals.both, method="fdr")  
    
  c(
    N1=N1, N2=N2, 
    TP.unadj=sum(pvals.sig < alpha), 
    FP.unadj=sum(pvals.nosig < alpha),
    TP.bh=sum(pvals.sig < alpha/(N1+N2)), 
    FP.bh=sum(pvals.nosig < alpha/(N1+N2)), 
    TP.fdr=sum(qvals.both[1:N1] < alpha), 
    FP.fdr=sum(qvals.both[-(1:N1)] < alpha)
  ) 
}

a simple example with both

  rbind(
    checkSignificance(1000, 1000, 0.05),
    checkSignificance(1000, 5000, 0.05),
    checkSignificance(1000, 10000, 0.05),
    checkSignificance(1000, 20000, 0.05)  
  )
